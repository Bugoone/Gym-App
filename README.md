# PeakForm 🏋️

A personalised gym tracking and workout generator app built for my 
A-Level Computer Science project.

## What it does

- Generates personalised workout plans based on user input
- Tracks workouts and stores data persistently
- Clean, intuitive interface built with Flet

## Built with

- Python
- Flet (UI framework)
- SQLite3

## How to run

1. Clone the repo
2. Install dependencies: "pip install flet"

3. Run the app: python main.py


## Project context

Built as part of my A-Level Computer Science coursework. Covers the full
development lifecycle — requirements gathering, design, implementation,
testing, and evaluation.
